# Safe Share

## Installation

```bash
# Clone the repository
python -m venv venv

## On linux
source venv/bin/activate

## On Windows
venv\Scripts\activate

cd safeshare
pip install -r requirements.txt
```

## Usage

```bash
# Run backend using docker-compose
docker-compose up
```

## Endpoints

- `/admin/` - Admin Panel
- `/api/` - API Root
